# Cmagic

cmagic is a tool for building Codemagic from CLI.

# Initial setup

add this tap:

```
brew tap imajoriri/cmagic
```

# Installing

To install the cmagic

```
brew install cmagic
```

# Updating

```
brew update cmagic
```

# Usage

### Start new build

```
cmagic start
```

options

- `-b, --branch`: Branch name to build against. Default is current branch.
